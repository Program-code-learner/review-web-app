🚀 Features
✅ View movie details
✅ Rate and review movies
✅ Store and manage data using MongoDB

🛠️ Tech Stack
Frontend: React, React Router, Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB
State Management: React Query
Forms & Validation: React Hook Form
